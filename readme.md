# Hybrid Tools

Hybrid Tools is an add-on package for the Hybrid Core WordPress framework. This package provides necessary tools for the framework.

## Requirements

* WordPress 4.9.6+.
* PHP 5.6+ (7.0+ recommended).
* [Composer](https://getcomposer.org/) for managing PHP dependencies.

## Documentation

This project is a part of the Hybrid Core framework. It may require other packages, which will be installed via Composer.

### Installation

Use the following command from your preferred command line utility to install the package.

```bash
composer require themehybrid/hybrid-tools
```
